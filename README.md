## A desktop-based Unit Converter application built using Java and Swing, allowing users to convert between multiple unit categories with a clean graphical interface and conversion history tracking.

Features

 ## Multiple Conversion Categories

Temperature (Celsius ↔ Fahrenheit)

Weight (Kilograms ↔ Pounds)

Length (Meters ↔ Feet)

 ## Graphical User Interface (GUI)

Built using Java Swing

Clean layout with intuitive controls

 ## Conversion History

View past conversions in a separate dialog

Includes input value, conversion type, and result

Option to clear history

 ## Smart UX Enhancements

Input field clears automatically when category or conversion changes

Units displayed clearly in results and history

Error handling for invalid inputs

## Technologies Used

Java

Java Swing (GUI)

Java Collections Framework

ArrayList

DefaultListModel

Object-Oriented Programming (OOP)

Event-driven programming
